地图 API 使用方式

本 HTML 默认可以直接打开：每个地点的小地图使用 Google Maps 嵌入链接加载；每日总路线会显示备用路线图和 Google Maps 路线按钮。若你要在页面中直接画出“步行/公交”的路线线条，请把下面脚本里的 YOUR_GOOGLE_MAPS_API_KEY 替换成你的 Google Maps Platform API Key，并启用 Maps JavaScript API。代码会用 DirectionsService 按每日分段绘制路线。

如何替换 API Key

搜索文件中的 const GOOGLE_MAPS_API_KEY = "YOUR_GOOGLE_MAPS_API_KEY";，替换成你的 Key。若 Key 受域名限制，本地双击 file:// 可能被拒，建议用 VS Code Live Server 或本地服务器打开。
